# Technology Stack

**Analysis Date:** 2026-04-14

## Languages

**Primary:**
- C# — all plugin logic in `src/ReservedSlots/ReservedSlots.cs`

**Secondary:**
- JSON — configuration and localization files (`lang/*.json`, generated config)

## Runtime

**Environment:**
- .NET 8.0 (`net8.0` target framework)
- Hosted inside the CounterStrikeSharp (CSS) plugin loader, which runs embedded within the CS2 game server process

**Package Manager:**
- NuGet (managed via `<PackageReference>` in `src/ReservedSlots/ReservedSlots.csproj`)
- Lockfile: not present (no `packages.lock.json`)

## Frameworks

**Core:**
- CounterStrikeSharp.API 1.0.365 — the CS2 server plugin framework; provides game event hooks, player/server utilities, admin management, localization, CVars, and timers

**Build/Dev:**
- MSBuild / `dotnet build` — standard .NET SDK build toolchain
- Build output: `src/ReservedSlots/bin/Debug/net8.0/ReservedSlots.dll`

**Testing:**
- None — no test framework or test files present

## Key Dependencies

**Critical:**
- `CounterStrikeSharp.API` 1.0.365 — the only direct dependency; transitively pulls in everything below

**Transitive (via CounterStrikeSharp.API):**
- `Microsoft.Extensions.Logging` 8.0.0 — used directly in plugin via `Logger.LogInformation` / `Logger.LogWarning`
- `Microsoft.Extensions.Localization.Abstractions` 8.0.3 — used directly via `Localizer["key"]` for i18n
- `Microsoft.Extensions.Hosting` 8.0.0 — plugin lifecycle (provided by CSS host)
- `McMaster.NETCore.Plugins` 1.4.0 — plugin loading mechanism inside CSS
- `Serilog.Extensions.Logging` 8.0.0 — logging backend
- `Serilog.Sinks.Console` 5.0.0 — console log output
- `Serilog.Sinks.File` 5.0.0 — file log output
- `Scrutor` 4.2.2 — DI scanning used by CSS
- `FastGenericNew` 3.3.1 — fast object creation used by CSS
- `Tomlyn` 0.19.0 — TOML config parsing used by CSS

**Plugin Config:**
- Automatically generated by CounterStrikeSharp at first run
- Location on server: `csgo/addons/counterstrikesharp/configs/plugins/ReservedSlots/ReservedSlots.json`
- Deserialized into `ReservedSlotsConfig : BasePluginConfig` via `IPluginConfig<ReservedSlotsConfig>`

**Environment:**
- No `.env` files or environment variables — all configuration is via the JSON config file and CSS admin flag system

**Build:**
- `src/ReservedSlots/ReservedSlots.csproj` — sole project file; no solution (`.sln`) file committed

## Platform Requirements

**Development:**
- .NET 8.0 SDK

**Production:**
- CS2 dedicated server with CounterStrikeSharp installed
